# Typing Battle

リアルタイムで対戦できる、マルチプレイヤー対応のタイピングバトルゲームです！
最大4人まで同じROOMに入り、タイピングの早さと正確さを競い合いながら、相手のHPを削って最後まで生き残ったプレイヤーが勝者となります。

## アプリケーションの概要
* **固定ロビー制**: `Room 1` から `Room 10` までの10個の部屋が用意されており、好きな部屋を選んで対戦できます。
* **実践的な日本語タイピング**: 表示される単語は「ことわざや慣用句」になっており、漢字（ルビ付き）とローマ字の両方が表示されます。
* **文字数ベースのダメージ計算**: 長いことわざを打ち切るほど、相手に与えるダメージが大きくなります（1文字あたり2ダメージ換算など）。
* **リアルタイム進捗バー**: 敵が今どれくらいタイピングを進めているかが、それぞれのプレイヤー名の下にプログレスバーとしてリアルタイムで表示され、白熱したバトルを演出します。

## 技術スタック
* **フロントエンド**: React (Vite), Vanilla CSS
* **バックエンド**: Node.js, Express, Socket.IO
* **インフラ**: Docker, Docker Compose

## セットアップと起動手順

DockerとDocker Composeがインストールされている環境であれば、以下のコマンド一発で起動できます。

### 1. アプリケーションの起動

プロジェクトのルートディレクトリ（`docker-compose.yml` がある階層）で、以下のコマンドを実行してください。

```bash
docker compose up -d --build
```
*※ 少し古いバージョンのDockerをご利用の場合は `docker-compose up -d --build` とハイフンを入れて実行してください。*

### 2. アプリへのアクセス

起動が完了したら、ブラウザを開いて以下のURLにアクセスします。

**http://localhost:5173**

### 3. オフにする場合

ゲームを終了し、コンテナを停止・削除する場合は以下のコマンドを実行します。

```bash
docker compose down
```

## 遊び方
1. **ロビー画面**: 「Your Name」に好きなプレイヤー名を入力し、空いているRoom（1〜10）の「JOIN」ボタンをクリックします。
2. **待機画面**: 参加した部屋の中で、2人以上プレイヤーが揃うのを待ちます。「READY TO FIGHT」をクリックして準備完了状態にします。
3. **対戦開始**: 全員がReady状態になり、かつ2名以上の参加者がいると「START BATTLE」ボタンが現れます。誰かがクリックすると対戦スタートです。
4. **バトル**: 画面に表示された「ローマ字」を正確にタイピングしてください。一文字打つごとに相手へのプレッシャーとなり、単語を完成させると相手全員のHPにダメージを与えます。自分のHPが0になったら脱落、最後の1人になったら勝利です！
