# Typing Battle

リアルタイムで対戦できる、マルチプレイヤー対応およびCPU対戦対応のタイピングバトルゲームです！
タイピングの早さと正確さを競い合いながら、相手のHPを削って勝利を目指します。

## アプリケーションの概要

### 1. マルチプレイヤーモード (Online Battle)
* **固定ロビー制**: `Room 1` から `Room 10` までの10個の部屋が用意されており、最大4人まで同時に対戦可能です。
* **文字数ベースのダメージ計算**: タイピングした単語の長さに応じて、相手に与えるダメージが変化します。
* **リアルタイム進捗**: 対戦相手のタイピング進捗がプログレスバーでリアルタイムに表示されます。

### 2. CPU対戦モード (CPU Battle)
* **1vs1バトル**: CPUと1対1でタイピングの腕を競います。
* **多彩なジャンル**: 「ことわざ」「ビジネス用語」「クラウド（19ジャンル）」の3カテゴリから選択可能です。
* **難易度選択**: Level 1 (C級) から Level 10 (Fast) まで、10段階の難易度が選べます。
* **ミス表示**: 間違ったキーを入力すると、該当文字が赤く点滅して正しいキーを促します。

## 技術スタック
* **フロントエンド**: React (Vite), Vanilla CSS
* **バックエンド**: Node.js, Express, Socket.IO
* **本番配信**: Nginx (静的ファイル配信 + リバースプロキシ)
* **タイピングエンジン**: 柔軟なローマ字入力（tu/tsu, si/shi, ttyu/cchuなど）を完全サポート
* **コンテナ**: Docker / Docker Compose

## セキュリティ対策
* CORS制限（`ALLOWED_ORIGIN` 環境変数で許可ドメインを指定）
* Socket.ioイベントの入力バリデーション（XSS対策、プレイヤー名のサニタイズ）
* タイピングイベントのレート制限（DoS対策）
* セキュリティヘッダ（CSP, X-Frame-Options, X-Content-Type-Options 等）
* 非rootコンテナ実行
* 本番環境ではNginxによる静的ファイル配信（Vite DevServer非公開）

## セットアップと起動手順

Docker環境があれば、以下のコマンドで全コンテナ（フロント・バック）を起動できます。

### 本番起動
```bash
docker compose up -d --build
```
ブラウザで **http://localhost** にアクセスします。

> **Note**: 外部公開時は `docker-compose.yml` の `ALLOWED_ORIGIN` を実際のドメインに変更してください。

### EC2などでのコンテナ自動起動設定（OS起動時）
Amazon Linux 2023などのサーバー運用において、インスタンス起動時に自動でアプリを立ち上げるには以下の設定を行います。

1. **Dockerサービスの自動起動を有効化**
   EC2が起動したときにDockerの土台が立ち上がるようにします。
   ```bash
   sudo systemctl enable docker
   sudo systemctl start docker
   ```
2. **コンテナの自動起動**
   `docker-compose.yml` には `restart: always` が設定されているため、Dockerサービスさえ立ち上がれば自動的にアプリのコンテナも起動します。
   そのため、初回に以下のコマンドでコンテナを作成しておけば以降は自動運用されます。
   ```bash
   docker compose up -d --build
   ```

### 開発起動（HMR・ホットリロード有効）
```bash
docker compose -f docker-compose.dev.yml up --build
```
ブラウザで **http://localhost** にアクセスします。

### 個別に起動する場合
**バックエンド:**
```bash
cd backend
npm install
npm run dev
```

**フロントエンド:**
```bash
cd frontend
npm install
npm run dev
```

## 遊び方
1. **モード選択**: トップ画面で「ONLINE BATTLE」または「CPU BATTLE」を選択します。
2. **ジャンル・難易度選択**: CPU戦の場合、カテゴリ（ことわざ/ビジネス用語/クラウド）を選択し、難易度を決定します。
3. **バトル開始**: 画面に表示される文字を正確にタイピングしてください。単語を完成させると相手のHPにダメージを与えます。間違ったキーを押すと文字が赤く光ります。自分のHPが尽きる前に相手のHPを0にすれば勝利です！
