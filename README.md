# Typing Battle

リアルタイムで対戦できる、マルチプレイヤー対応およびCPU対戦対応のタイピングバトルゲームです！
タイピングの早さと正確さを競い合いながら、相手のHPを削って勝利を目指します。

## アプリケーションの概要

### 1. マルチプレイヤーモード (Online Battle)
* **固定ロビー制**: `Room 1` から `Room 10` までの10個の部屋が用意されており、最大4人まで同時に対戦可能です。
* **文字数ベースのダメージ計算**: タイピングした単語の長さに応じて、相手に与えるダメージが変化します。
* **リアルタイム進捗**: 対戦相手のタイピング進捗がプログレスバーでリアルタイムに表示されます。

### 2. CPU対戦モード (CPU Battle) - **NEW!**
* **1vs1バトル**: CPUと1対1でタイピングの腕を競います。
* **多彩なテクニカルジャンル**: 従来の「ことわざ」に加え、インフラ・ネットワークを中心とした19の専門的な技術ジャンルを選択可能です。
* **難易度選択**: Level 1 (C級) から Level 10 (Fast) まで、自分のレベルに合わせた10段階の難易度が選べます。

## 技術スタック
* **フロントエンド**: React (Vite), Vanilla CSS
* **バックエンド**: Node.js, Express, Socket.IO

## セットアップと起動手順

Docker環境があれば、以下のコマンドで全コンテナ（フロント・バック）を起動できます。

### 1. アプリケーションの起動
```bash
docker compose up -d --build
```

### 2. アプリへのアクセス
ブラウザで以下のURLを開きます。
**http://localhost:5173**

### 3. 個別に起動する場合 (Development)
**バックエンド:**
```bash
cd backend
npm install
npm run dev
```

**フロントエンド:**
```bash
cd frontend
npm install
npm run dev
```

## 遊び方
1. **モード選択**: トップ画面で「Battle Online」または「Battle CPU」を選択します。
2. **ジャンル・難易度選択**: CPU戦の場合、カテゴリ（ことわざ/クラウド）とジャンルを選択後、難易度を決定します。
3. **バトル開始**: 画面に表示される文字を正確にタイピングしてください。一文字打つごとにダメージが蓄積され、単語を完成させると相手のHPに大きなダメージを与えます。自分のHPが尽きる前に相手のHPを0にすれば勝利です！
